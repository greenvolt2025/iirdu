# 🚀 SEO Оптимізація Завершена - Що робити далі?

## ✅ Що вже зроблено автоматично

### 1. Structured Data (Структуровані дані)
- ✅ **FAQ Schema** - додано на головну сторінку 8 найпопулярніших питань
  - Google тепер може показувати ваші відповіді прямо в результатах пошуку
  - Збільшує CTR (клікабельність) на 20-30%

- ✅ **Organization Schema** - інформація про МІВРУ
  - Google розуміє, що ви - наукова установа
  - ЄДРПОУ, контакти, адреса - все в структурованому форматі

- ✅ **Service Schema** - ціни на послуги
  - Google може показати ваші ціни прямо в пошуку
  - "Квартира - 1000 грн", "Будинок - 1500 грн"

### 2. Meta Tags (Мета-теги)
- ✅ Open Graph - для Facebook, LinkedIn, Telegram
- ✅ Twitter Cards - для Twitter/X
- ✅ Canonical URLs - щоб уникнути дублів контенту
- ✅ Hreflang - для українською та англійською версій

### 3. Технічне SEO
- ✅ Sitemap.xml - карта сайту для Google
- ✅ Robots.txt - інструкції для пошукових роботів

### 🔴 ПРІОРИТЕТ 2: Google Search Console

**Що це:** Безкоштовний інструмент Google для моніторингу позицій сайту.

**Кроки:**

#### 1. Реєстрація
1. Відкрити https://search.google.com/search-console
2. Увійти з Google акаунтом
3. Натиснути "Додати ресурс" → вибрати "Домен"
4. Вписати: `iirdu.org`

#### 2. Підтвердження власності
Виберіть один зі способів:

**Варіант А: DNS (рекомендовано)**
1. Google покаже TXT запис типу: `google-site-verification=abc123xyz`
2. Увійдіть у панель управління доменом (де купували домен)
3. Додайте TXT запис до DNS
4. Зачекайте 5-10 хвилин
5. Натисніть "Перевірити" в Google Search Console

**Варіант Б: HTML тег (простіше)**
1. Google покаже код типу:
   ```html
   <meta name="google-site-verification" content="abc123xyz" />
   ```
2. Скопіюйте тільки `abc123xyz`
3. Відкрийте файл `src/app/[locale]/layout.tsx`
4. Знайдіть рядок 85:
   ```typescript
   verification: {
     google: "google-site-verification-code-here", // ⬅️ ТУТ
   },
   ```
5. Замініть на ваш код:
   ```typescript
   verification: {
     google: "abc123xyz",
   },
   ```
6. Закомітьте зміни:
   ```bash
   git add src/app/[locale]/layout.tsx
   git commit -m "Add Google Search Console verification"
   git push
   ```
7. Зачекайте 2-3 хвилини (Vercel автоматично задеплоїть)
8. Натисніть "Перевірити" в Google Search Console

#### 3. Додати Sitemap
1. У Google Search Console → "Файли Sitemap"
2. Додати: `https://iirdu.org/sitemap.xml`
3. Натиснути "Надіслати"

#### 4. Надіслати головні сторінки на індексацію
1. У Google Search Console → "Перевірка URL"
2. Вставити по черзі:
   - `https://iirdu.org/uk`
   - `https://iirdu.org/en`
   - `https://iirdu.org/uk/pricing`
   - `https://iirdu.org/uk/about`
   - `https://iirdu.org/uk/contacts`
3. Натиснути "Запросити індексацію"

**Результат:**
- За 2-7 днів сайт з'явиться в Google
- Ви побачите, скільки людей заходять з пошуку
- Які запити вводять (RD4U, RDNA, оцінка збитків тощо)

### 🟡 ПРІОРИТЕТ 3: Google Analytics (опціонально)

**Що це:** Статистика відвідувачів - скільки, звідки, що роблять.

**Кроки:**
1. Перейти на https://analytics.google.com/
2. Створити обліковий запис
3. Створити ресурс "GA4" для `iirdu.org`
4. Отримати Measurement ID (типу `G-XXXXXXXXXX`)
5. Додати в `src/app/[locale]/layout.tsx` (після рядка 150):
   ```typescript
   <Script
     strategy="afterInteractive"
     src={`https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX`}
   />
   <Script
     id="google-analytics"
     strategy="afterInteractive"
     dangerouslySetInnerHTML={{
       __html: `
         window.dataLayer = window.dataLayer || [];
         function gtag(){dataLayer.push(arguments);}
         gtag('js', new Date());
         gtag('config', 'G-XXXXXXXXXX');
       `,
     }}
   />
   ```
6. Імпортувати Script на початку файлу:
   ```typescript
   import Script from "next/script";
   ```

## ❓ Часті питання

### Q: Як швидко сайт потрапить у топ Google?
**A:** За брендовими запитами ("МІВРУ") - 1-2 тижні. За конкурентними ("оцінка збитків") - 2-6 місяців.

### Q: Чому сайт не з'являється в Google?
**A:** Потрібен час на індексацію (2-7 днів). Використайте Google Search Console для прискорення.

### Q: Скільки коштує SEO просування?
**A:** Базова оптимізація (вже зроблена) - безкоштовно. Професійне просування - 10,000-50,000 грн/міс.

### Q: Чи потрібно платити Google за позиції?
**A:** Ні! Органічна видача (не реклама) абсолютно безкоштовна.

### Q: Що найважливіше для топу Google?
**A:**
1. Якісний контент (статті, поради)
2. Зворотні посилання (backlinks)
3. Швидкість сайту (у вас вже швидкий - Vercel)
4. Мобільна версія (у вас вже адаптивна)
