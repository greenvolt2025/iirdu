## Код для mailto генерації

### Проста CTA:
```tsx
<a href={`mailto:${siteConfig.email}?subject=${encodeURIComponent("Тема")}`}>
  <Button>Написати</Button>
</a>
```

### З попередньо заповненим body:
```tsx
const body = `Доброго дня!\n\nПрошу підготувати висновок.\n\nДякую!`;
<a href={`mailto:${siteConfig.email}?subject=${encodeURIComponent(subject)}&body=${encodeURIComponent(body)}`}>
  <Button>Написати</Button>
</a>
```
